# Start Fixer

Use this skill when a fresh GitHub-ready Fixer session needs the project-scoped orchestrator role.

Checklist:
- confirm the repo boundary is `github_repo/`
- inspect attached docs and active MCP assignments
- re-anchor on the temporary flow policy: only `manual-resolution` and `autonomous-resolution-one-task` are allowed
- treat parallel Netrunner orchestration and autonomous fixer resume loops as disabled in this track
- initialize only the Fixer checklist first
- stop after initialization and report status
